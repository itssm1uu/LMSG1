Proyecto 2: CSS

Mi nombre es Miruna G. Matei y en este proyecto presento una página web dedicada a ofrecer información sobre CSS. Exploro los conceptos fundamentales de las hojas de estilo en cascada, sus propiedades, selectores y cómo permiten diseñar y personalizar la apariencia de las páginas web. Esta página busca acercar al público los beneficios y posibilidades que ofrece CSS en el desarrollo web moderno.
